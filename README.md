# Sistema de Login em PHP

Sistema completo de autenticação desenvolvido com PHP e MySQL.

# Funcionalidades

* Cadastro de usuário
* Login com senha criptografada
* Sessão de usuário
* Dashboard protegido
* Logout
* Recuperação de senha
* Mensagens de erro e sucesso

# Tecnologias utilizadas

* PHP
* MySQL
* HTML
* CSS
* Sessions
* Password Hash

# Como executar

# Rodar localmente

1. Instale o XAMPP
2. Coloque o projeto na pasta htdocs
3. Configure o banco de dados MySQL
4. Atualize os dados no arquivo conexao.php
5. Acesse localhost no navegador

# Rodar online
